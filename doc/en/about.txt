YouTube Live Chat Client
Version: 1.7 (2026.06.06)

ABOUT
This add-on allows visually impaired users to track the YouTube live stream chat flow, Super Chat donations, Super Sticker sends, and channel membership notifications via the NVDA screen reader through a fully accessible interface. 
Fully compatible with NVDA 2026.1 and later 64-bit versions, this client fetches data silently in the background, ensuring an uninterrupted experience without affecting your system performance. 
It includes advanced features such as automatically reading new incoming messages aloud, adding timestamps to messages, automatic focus on the list, and customizable API request intervals. 
Additionally, it has comprehensive multi-language support, including Turkish, English, and German.
