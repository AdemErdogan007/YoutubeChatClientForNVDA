YouTube Live Chat Client

This plugin allows you to track chat messages, Super Chat donations, and membership events in YouTube live broadcasts with an accessible interface.

Installation and First Steps

  1. Restart NVDA after installing the plug-in.
  2. Run the plug-in by pressing NVDA + Shift + CTRL + Y or from the NVDA Tools menu.
  3. You will be asked to enter API Key at first startup. It will automatically redirect you to the Settings window.
  4. Paste the YouTube Data v3 API key you received from the Google Cloud Console here and press the "Save Settings" button.

Main Window Usage

  • URL Field: Paste the link of the publication you want to follow here.
  • Options Menu: You can access Settings, Version History, API Guide, About and Website buttons from this menu.
  • Start/Stop: Controls the chat flow.
  • Chat List: Messages fall here. You can navigate with the arrow keys.

Settings Menu

  • API Key: Appears with an asterisk for security reasons. You can see the content with the "Show" button and clear it with the "Delete" button.
  • Timestamp: Determines whether the time will be written at the beginning of the messages.
  • Automatic Voice: Determines whether NVDA will automatically read a new message when it arrives.
  • Move Focus to Last Message: Important! If you are reviewing old messages, turn this off.
    If it is on, the cursor moves to the bottom with each new message.
  • API Request Duration: Determines how many seconds data will be retrieved from YouTube (Default 10 seconds).
  • Reset to Defaults:Resets all settings to initial setup.
