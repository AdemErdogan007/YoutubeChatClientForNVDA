YouTube Live-Chat-Client

Version : 1.7 (2026.06.06)

Entwickler : Adem ERDOĞAN

Urheberrecht : Copyright © 2026, Adem ERDOĞAN (Erdoğan Teknoloji ve Programcılık).

Dieses Add-on ermöglicht es sehbehinderten Benutzern, den Chat-Verlauf, Super Chat-Spenden und Kanalmitgliedschafts-Benachrichtigungen in YouTube-Livestreams über den NVDA-Screenreader über eine 100 % barrierefreie Oberfläche zu verfolgen.
Vollständig kompatibel mit NVDA 2026.1 und neueren 64-Bit-Versionen.

Installation und erste Schritte

  1. Starten Sie NVDA nach der Installation des Add-ons neu.
    Das Add-on erkennt die Sprache von NVDA (Türkisch, Englisch oder Deutsch) automatisch.
  2. Drücken Sie die Tasten NVDA + Umschalt + STRG + Y oder starten Sie das Add-on über das NVDA-Menü "Werkzeuge".
  3. Beim ersten Start werden Sie aufgefordert, einen API-Schlüssel einzugeben.
    Das System leitet Sie automatisch zum Einstellungsfenster weiter.
  4. Fügen Sie hier den von der Google Cloud Console erhaltenen YouTube Data v3 API-Schlüssel ein und klicken Sie auf die Schaltfläche "Einstellungen speichern".

Wie erhält man einen API-Schlüssel? (Detaillierte Anleitung)

Diese Anleitung erklärt Schritt für Schritt, wie Sie den kostenlosen API-Schlüssel erhalten, den das Add-on benötigt, um sich mit YouTube-Chats zu verbinden.
Auch wenn Sie keine technischen Kenntnisse haben, können Sie Ihren Schlüssel ganz einfach erstellen, indem Sie diese Schritte der Reihe nach befolgen.
Hinweis : Teilen Sie Ihren API-Schlüssel mit niemandem.

Schritt 1: Bei der Google Cloud Platform anmelden

  1. Klicken Sie im Add-on auf die Schaltfläche "Google Cloud Console im Browser öffnen" oder gehen Sie in Ihrem Browser auf https://console.cloud.google.com .
  2. Melden Sie sich mit einem bestehenden Google-Konto (Gmail) an.
    Wenn Sie sich zum ersten Mal am System anmelden, wird möglicherweise eine Nutzungsvereinbarung oder ein Begrüßungsbildschirm angezeigt. Bestätigen Sie diese, um fortzufahren.

Schritt 2: Ein neues Projekt erstellen

  1. Klicken Sie oben auf dem Bildschirm auf die Schaltfläche "Projekt auswählen" (Select a project). Suchen und klicken Sie oben rechts im geöffneten Fenster auf die Schaltfläche "Neues Projekt" (New Project).
  2. Geben Sie im Feld "Projektname" (Project Name) einen einfachen Namen ein, an den Sie sich für das Add-on erinnern können (Zum Beispiel: "NVDA Youtube Chat").
  3. Klicken Sie unten auf die Schaltfläche "Erstellen" (Create). Das Erstellen des Projekts kann einige Sekunden dauern. Warten Sie, bis es abgeschlossen ist.

Schritt 3: Die YouTube API aktivieren

  1. Klicken Sie nach dem Erstellen Ihres Projekts im linken Navigationsmenü auf die Option "APIs & Dienste" (APIs & Services) und dann auf die Registerkarte "Bibliothek" (Library).
  2. Geben Sie "YouTube Data API v3" in das angezeigte Suchfeld ein und suchen Sie danach.
  3. Klicken Sie in den Suchergebnissen auf den Link "YouTube Data API v3".
  4. Klicken Sie auf der geöffneten Seite auf die Schaltfläche "Aktivieren" (Enable).
    Dadurch erhält Ihr erstelltes Projekt die Berechtigung, YouTube-Live-Chat-Daten abzurufen.

Schritt 4: Den API-Schlüssel generieren

  1. Klicken Sie nach der Aktivierung der API auf der Seite, auf die Sie weitergeleitet wurden, oder im linken Menü auf die Registerkarte "Anmeldedaten" (Credentials).
  2. Klicken Sie oben auf dem Bildschirm auf die Schaltfläche "Anmeldedaten erstellen" (Create Credentials).
  3. Wählen Sie aus dem kleinen Dropdown-Menü die Option "API-Schlüssel" (API key).

Schritt 5: Den Schlüssel kopieren und dem Add-on hinzufügen

  1. Auf dem Bildschirm öffnet sich ein Fenster mit der Meldung "API-Schlüssel erstellt", und ein komplexer, langer Code, der mit den Buchstaben "AIza" beginnt, wird angezeigt.
  2. Dieser lange Text ist Ihr API-Schlüssel.
    Kopieren Sie ihn, indem Sie auf die Schaltfläche "Kopieren" neben dem Text klicken oder den Text vollständig markieren.
  3. Kehren Sie nun zu den NVDA-Add-on-Einstellungen zurück, fügen Sie diesen kopierten Schlüssel in das Feld "Google API-Schlüssel" ein und speichern Sie die Einstellungen.

WICHTIGER SICHERHEITSHINWEIS:
Dieser Schlüssel ist ganz für Sie persönlich. Bitte teilen Sie ihn nicht mit anderen oder auf öffentlichen Plattformen.
Google bietet diesen Dienst für den persönlichen Gebrauch völlig kostenlos an, und das zugewiesene tägliche Kontingent zum Abrufen von Nachrichten reicht für den persönlichen Gebrauch völlig aus.

Verwendung des Hauptfensters

  • URL-Feld: Fügen Sie hier den Link des YouTube-Livestreams ein, den Sie verfolgen möchten.
  • Optionen-Menü: Über dieses Menü können Sie schnell auf alle Unterfenster wie Einstellungen, Versionsverlauf, API-Anleitung, Über und Webseite zugreifen.
  • Start/Stopp: Startet oder stoppt den Chat-Verlauf.
    Der Verlauf läuft im Hintergrund, ohne NVDA zu belasten.
  • Chat-Liste: Eingehende Nachrichten erscheinen hier in einer barrierefreien Liste.
    Sie können mit den Pfeiltasten zwischen den Nachrichten navigieren.

Einstellungsmenü

  • API-Schlüssel: Wird aus Sicherheitsgründen maskiert (mit Sternchen) angezeigt.
    Sie können den Inhalt mit der Schaltfläche "Anzeigen" lesen und mit der Schaltfläche "Löschen" sofort löschen.
  • Zeitstempel: Legt fest, ob die Sendezeit am Anfang der Nachrichten geschrieben wird.
  • Automatisches Vorlesen: Steuert, ob der Screenreader die Nachricht automatisch vorliest, wenn eine neue Nachricht eintrifft.
  • Fokus auf letzte Nachricht verschieben: Wichtig! Wenn Sie in der Liste nach oben scrollen, um ältere Nachrichten zu lesen, schalten Sie diese Einstellung aus.
    Wenn sie aktiviert ist, wird der Fokus des Screenreaders bei jeder neuen eingehenden Nachricht zwingend an das Ende der Liste verschoben.
  • API-Abfrageintervall: Legt fest, alle wie viel Sekunden Daten von den YouTube-Servern abgerufen werden (Standard ist 10 Sekunden).
    Wenn Sie diesen Wert zu niedrig ansetzen, ist Ihr tägliches API-Limit sehr schnell aufgebraucht.
    Es wird empfohlen, einen Wert im Bereich von 5 bis 15 Sekunden zu verwenden.
  • Auf Standard zurücksetzen: Setzt alle vorgenommenen Änderungen zurück und versetzt die Einstellungen wieder in ihren ursprünglichen Installationszustand.
