YouTube Live-Chat-Client

Dieses Add-on ermöglicht es Ihnen, Chat-Nachrichten, Super Chat-Spenden und Mitgliedschaftsereignisse in YouTube-Livestreams über eine barrierefreie Schnittstelle zu verfolgen.

Installation und erste Schritte

  1. Starten Sie NVDA nach der Installation des Add-ons neu.
  2. Starten Sie das Add-on, indem Sie die Tasten NVDA + Umschalt + STRG + Y drücken oder es über das NVDA-Menü "Extras" aufrufen.
  3. Beim ersten Start werden Sie aufgefordert, einen API-Schlüssel einzugeben. Sie werden automatisch zum Einstellungsfenster weitergeleitet.
  4. Fügen Sie Ihren YouTube Data v3 API-Schlüssel von der Google Cloud Console hier ein und klicken Sie auf die Schaltfläche "Einstellungen speichern" .

Verwendung des Hauptfensters

  • URL-Feld: Fügen Sie hier den Link des Streams ein, den Sie verfolgen möchten.
  • Menü "Optionen": Über dieses Menü können Sie auf Einstellungen, Versionsverlauf, API-Leitfaden, Über und die Website zugreifen.
  • Start/Stopp: Steuert den Chat-Verlauf (Stream).
  • Chat-Liste: Nachrichten erscheinen hier. Sie können mit den Pfeiltasten darin navigieren.

Einstellungsmenü

  • API-Schlüssel: Wird aus Sicherheitsgründen mit Sternchen angezeigt. Sie können den Inhalt mit der Schaltfläche "Anzeigen" sehen oder mit "Löschen" entfernen.
  • Zeitstempel: Legt fest, ob die Uhrzeit am Anfang der Nachrichten angezeigt wird.
  • Automatisches Vorlesen: Legt fest, ob NVDA neue Nachrichten automatisch vorliest, sobald sie eintreffen.
  • Fokus auf letzte Nachricht verschieben: Wichtig! Schalten Sie dies aus, wenn Sie alte Nachrichten durchsehen möchten.
    Wenn aktiviert, springt der Cursor bei jeder neuen Nachricht nach ganz unten.
  • API-Abfrageintervall: Legt fest, alle wie viele Sekunden Daten von YouTube abgerufen werden (Standard ist 10 Sekunden).
  • Auf Standardwerte zurücksetzen: Setzt alle Einstellungen auf den Zustand der Erstinstallation zurück.
